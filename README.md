# VibingWithJen

A simple Python calculator with a web-based UI running on localhost, built to learn Cline and basic web development with minimal dependencies.

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Calculation history display
- Clean, simple web interface
- Runs locally with Flask

## Tech Stack

- Python 3.12
- Flask (web framework)
- HTML/CSS/JavaScript for frontend

## Memory Bank

This project maintains a memory-bank directory for technical documentation and context. See `memory-bank/tech_context.md` for detailed technical information, decisions, and project state.

## Getting Started

1. Install Flask: `pip install flask`
2. Run the app: `python app.py`
3. Open http://localhost:5000 in your browser

## Project Structure

- `README.md` - This file
- `memory-bank/tech_context.md` - Technical context and documentation
- `app.py` - Main Flask application
- `templates/` - HTML templates
